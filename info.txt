    Тестовое задание по генерации текстового отчета. Отчет выводится в консоль.

https://github.com/usilitel/ParserTest_v2
    Парсер текстового файла.
    - Принимает на вход файл с древовидной структурой
    - разбирает его по узлам
    - записывает их в массив
    - выводит данные в текстовый файл такой структуры: (id узла, id вышестоящего узла, имя узла, значение в узле).

https://github.com/usilitel/Intervals
    - Найти и вывести в интерфейсе пересечение заданных множеств вещественных чисел.
    - Найти число, принадлежащее пересечению подмножеств, максимально близкое к заданному x.

https://github.com/usilitel/Dinners
    Программа заказа обедов в офис. Исходные данные хранятся в формате xml.
    Результат работы программы - два отчета:
    Первый для отправки заказа в столовую, второй для раздачи заказанных блюд сотрудникам.
        Первый отчет состоит из таблицы со столбцами: название блюда, количество, стоимость. После таблицы должна быть напечатана полная стоимость заказа.
        Второй отчет состоит из таблицы со столбцами: имя сотрудника, заказанные блюда, стоимость его обеда.
